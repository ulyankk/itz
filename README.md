# Фильтрация учебной нагрузки

## Описание проекта

Данный проект предназначен для автоматического переноса строк с обычной учебной нагрузкой из сводного файла прошлого года в новый файл текущего года. Скрипт исключает нагрузки, связанные с дипломами, руководством, практикой, ГИА и другими служебными видами работ.

## Задача

Из исходного файла `Нагрузка.xlsx` (лист "общее") необходимо отфильтровать строки, оставив только обычную учебную нагрузку, и сохранить результат в новый файл.

## Используемые технологии

- Python 3
- Pandas

## Установка и запуск

1. Установить зависимости:
```bash
pip install pandas openpyxl
```
2. Поместить файл Нагрузка.xlsx в папку ../data/ (относительно скрипта) или изменить путь PATH_DATA в коде

3. Запустить скрипт:

```bash
python filter_load.py
```
## Результат
Исходное количество строк: 1164

Отфильтровано строк с обычной нагрузкой: 909

Результат сохранён в файл: новый_сводный_файл_2026.xlsx
