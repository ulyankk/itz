# 📊 Фильтрация учебной нагрузки

> Автоматический перенос строк с обычной учебной нагрузкой из сводного файла прошлого года в новый файл текущего года

---

## 📋 О проекте

Скрипт предназначен для обработки Excel-файлов с учебной нагрузкой. Он фильтрует строки, исключая все виды нагрузок, связанные с **дипломами**, **руководством**, **практикой**, **ГИА** и другими служебными видами работ, и сохраняет результат в новый файл.

## 🚀 Установка и запуск

### 1. Установить зависимости:
```bash
pip install pandas openpyxl
```
Поместить файл Нагрузка.xlsx в папку ../data/ (относительно скрипта) или изменить путь PATH_DATA в коде

### 2. Запустить скрипт:

bash
python filter_load.py
```
Результат
Исходное количество строк: 1164

Отфильтровано строк с обычной нагрузкой: 909

Результат сохранён в файл: новый_сводный_файл_2026.xlsx
